Multi-Centroid Hyperdimensional Computing for Compact IMC Arrays via Dimension Pruning

IEEE Internet of Things Journal (Under Review)

Authors

Do Yeong Kang, Yeong Hwan Oh, Chanwook Hwang, Jinhee Kim, Kang Eun Jeon*, Jong Hwan Ko*

Abstract

The implementation of Hyperdimensional Computing (HDC) on In-Memory Computing (IMC) architectures faces fundamental challenges due to dimensional mismatches. This paper presents MEMHD, a framework that uses multiple class vectors to achieve full IMC array utilization, and DiP-MEMHD, which prunes dimensions for compact arrays. MEMHD achieves 13.25× better memory efficiency, and DiP-MEMHD shows 2.09% higher accuracy than existing pruning metrics and achieves 22.4% higher accuracy than state-of-the-art binary HDC models on compact IMC arrays with 6-8× better energy efficiency.

Keywords

Hyperdimensional Computing, In-Memory Computing, Multi-Centroid Associative Memory, Dimension Pruning

MEMHD: Memory-Efficient Multi-Centroid Hyperdimensional Computing for Fully-Utilized In-Memory Computing Architectures

DATE 2025

Authors

Do Yeong Kang, Yeong Hwan Oh, Chanwook Hwang, Jinhee Kim, Kang Eun Jeon*, Jong Hwan Ko*

Abstract

This paper presents MEMHD, a framework to address the mismatch between high-dimensional vectors and IMC array sizes. MEMHD introduces clustering-based initialization and quantization-aware learning for multi-centroid associative memory, achieving full IMC array utilization and one-shot associative search. It outperforms state-of-the-art binary HDC models, achieving up to 13.69% higher accuracy or 13.25x more memory efficiency, and reduces computation cycles by up to 80x.

Keywords

Hyperdimensional Computing, In-Memory Computing, Multi-Centroid
