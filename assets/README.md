# assets

Put your files here:

- `photo.jpg` — a square profile photo (e.g. 400×400). Referenced by the hero once you swap the
  placeholder `<div class="photo">` for an `<img>` (see the root README).
- `cv.pdf` — your CV. The hero button and the CV section link to `assets/cv.pdf`.

Until you add these, the photo shows initials and the CV links will 404 — that's expected for the scaffold.
